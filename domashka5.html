del - Помечает текст как удаленный, перечёркивая его.

ins - Выделяет текст подчеркиванием. Применяется для выделения изменений, вносимых в документ.

smal - Отображает текст шрифтом меньшего размера.

b - Задает полужирное начертание отрывка текста, не придавая акцент или важность выделенному.

strong - Чтобы текст внутри тега стал жирным.

sup - Задает надстрочное написание символов. Например, 32

sub - Задает подстрочное написание символов, например, индекса элемента в химических формулах. Например, H2SO4

code - Представляет фрагмент программного кода, отображается шрифтом семейства monospace.

pre - Выводит текст без форматирования, с сохранением пробелов и переносов текста. Может быть использован для отображения компьютерного кода, сообщения электронной почты и т.д.kovalevasperm@gmail.com

span - Контейнер для строчных элементов. Можно использовать для форматирования отрывков текста, например, выделения цветом отдельных слов. Например,style="color: blue
